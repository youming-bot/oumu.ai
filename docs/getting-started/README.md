# Getting Started

Welcome to the Shadowing Learning project! This guide will help you get up and running quickly, whether you're a new user or a developer joining the project.

## 🚀 Quick Start

### What You'll Need
- **For Users**: Modern web browser (Chrome, Firefox, Safari, Edge)
- **For Developers**: Node.js 18+, npm or pnpm, Git

### Quick Setup

#### For Users (Deployment)
1. Visit the [live application](https://your-app-url.com)
2. Upload an audio file (MP3, WAV, M4A)
3. Wait for transcription to complete
4. Start learning with synchronized subtitles!

#### For Developers (Local Development)
```bash
# Clone the repository
git clone https://github.com/your-repo/shadowing-learning.git
cd shadowing-learning

# Install dependencies
npm install

# Configure environment variables
cp .env.example .env.local
# Edit .env.local with your API keys

# Start development server
npm run dev

# Open http://localhost:3000
```

## 📋 Prerequisites

### System Requirements
- **Operating System**: Windows 10+, macOS 10.15+, Linux
- **Node.js**: Version 18.0 or higher
- **Memory**: Minimum 4GB RAM (8GB recommended)
- **Storage**: 500MB free space

### Required API Keys
You'll need API keys from:
- **Groq**: For speech-to-text transcription
- **OpenRouter**: For text processing and normalization

### Browser Support
- Chrome 90+
- Firefox 88+
- Safari 14+
- Edge 90+
