## 💡 Learning Philosophy

Shadowing learning is based on the principle of **active listening and repetition**:

1. **Listen**: Hear native pronunciation and rhythm
2. **Shadow**: Repeat immediately after the speaker
3. **Compare**: Notice differences and adjust
4. **Repeat**: Practice until natural

This application provides the tools to make this process efficient and effective.
