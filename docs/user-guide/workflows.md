## 🚀 Beginner Workflows

### Basic Learning Workflow

#### Overview
Perfect for new users to learn the core shadowing technique.

#### Time Required
- **Setup**: 5 minutes
- **Practice Session**: 15-30 minutes
- **Review**: 10 minutes

#### Steps

1. **Choose Your Content**
   - Select short, clear audio (1-3 minutes)
   - Content should be at your level or slightly above
   - Good options: podcasts, audiobooks, news clips

2. **Upload and Process**
   ```bash
   # Drag and drop file
   # Wait for processing to complete
   # Usually takes 2-5 minutes
   ```

3. **Initial Listen-Through**
   - Play the entire audio once
   - Read along with subtitles
   - Don't stop or repeat yet
   - Focus on overall meaning

4. **Sentence-by-Sentence Practice**
   - Play one sentence
   - Pause immediately after
   - Shadow (repeat) exactly what you heard
   - Continue to next sentence

5. **A-B Loop Difficult Parts**
   - Identify difficult sentences
   - Set A and B loop points
   - Repeat 5-10 times
   - Focus on pronunciation and rhythm

6. **Review and Reflect**
   - Note difficult words/phrases
   - Add to vocabulary if needed
   - Practice again the next day

#### Tips for Success
- **Start Small**: 1-2 minute segments are perfect
- **Be Consistent**: Daily 15-minute sessions are better than weekly marathons
- **Record Yourself**: Compare your pronunciation with the original
- **Patience**: Progress takes time, celebrate small improvements

### Simple Shadowing Practice

#### Overview
Basic shadowing technique for language learners.

#### What You'll Need
- Processed audio file with subtitles
- Quiet environment
- Headphones (recommended)
- 15-30 minutes of focused time

#### Steps

1. **Preparation (2 minutes)**
   - Find a quiet space
   - Put on headphones
   - Open the processed file
   - Set volume to comfortable level

2. **Full Listening (3 minutes)**
   - Play entire audio once
   - Just listen, don't speak
   - Focus on rhythm and intonation
   - Follow along with subtitles

3. **Sentence Shadowing (10 minutes)**
   ```
   For each sentence:
   1. Listen to original
   2. Shadow immediately after
   3. Compare your pronunciation
   4. Repeat if necessary
   ```

4. **Difficult Part Practice (5 minutes)**
   - Identify 2-3 difficult phrases
   - Use A-B loop to repeat them
   - Practice until comfortable

5. **Full Run-through (2 minutes)**
   - Play entire audio again
   - Shadow along continuously
   - Notice improvements from step 2

#### Progress Tracking
- Record yourself before and after
- Note difficult words for later review
- Keep a practice log
- Set goals for next session

## 🎓 Intermediate Workflows

### Vocabulary Building Workflow

#### Overview
Systematic approach to building vocabulary from audio content.

#### Time Required
- **Initial Setup**: 10 minutes
- **Daily Practice**: 20-30 minutes
- **Review Sessions**: 15 minutes

#### Steps

1. **Content Selection**
   - Choose content rich in target vocabulary
   - Audio should be at appropriate difficulty level
   - Topics should interest you

2. **Processing and Analysis**
   - Upload and process audio
   - Review generated subtitles
   - Identify unknown words/phrases

3. **Vocabulary Extraction**
   - Use built-in terminology features
   - Manually select important words
   - Add context sentences
   - Categorize by topic/difficulty

4. **Study Session Structure**
   ```
   5 minutes: Review previous vocabulary
   10 minutes: Learn new words from current content
   5 minutes: Practice in context
   5 minutes: Quick quiz/review
   ```

5. **Spaced Repetition**
   - Review words after 1 day, 3 days, 1 week
   - Use built-in flashcards or export to Anki
   - Track retention rates
   - Adjust study frequency

6. **Integration with Practice**
   - Focus on vocabulary words during shadowing
   - Create example sentences
   - Practice in different contexts

#### Tools and Methods
- **Built-in Tools**: Terminology glossary, flashcards
- **Export Options**: Anki, Quizlet, text files
- **Practice Methods**: Contextual usage, role-play
- **Tracking**: Progress charts, retention metrics

### Multi-language Study Workflow

#### Overview
Study multiple languages simultaneously using the app.

#### Language Pairing Strategies
- **Related Languages**: Spanish + Italian
- **Language Families**: Germanic languages, Romance languages
- **Skill Complementary**: Listening practice + reading practice
- **Time Management**: Primary vs. secondary languages

#### Implementation Strategy

##### Daily Schedule Example
```
Morning (30 min): Primary language (Japanese)
- Process new content
- Vocabulary building
- Shadowing practice

Afternoon (20 min): Secondary language (Spanish)
- Review previous material
- Listening comprehension
- Quick vocabulary session

Evening (15 min): Language maintenance
- Mixed content from both languages
- Review difficult concepts
- Light practice
```

##### Weekly Rotation
```
Week 1: Focus on Japanese (70%) / Spanish (30%)
Week 2: Balanced approach (50% / 50%)
Week 3: Focus on Spanish (70%) / Japanese (30%)
Week 4: Integration and review
```

#### Content Management
1. **Language Organization**
   - Separate content by language
   - Use tags and categories
   - Create language-specific playlists
   - Track progress separately

2. **Cross-Language Activities**
   - Compare similar concepts
   - Find cognates and false friends
   - Practice language switching
   - Create translation exercises

3. **Resource Allocation**
   - Balance time between languages
   - Set appropriate difficulty levels
   - Choose complementary content
   - Monitor overall language load

#### Challenges and Solutions
- **Interference**: Practice languages at different times
- **Time Management**: Use micro-sessions throughout day
- **Motivation**: Set small, achievable goals
- **Progress Tracking**: Use separate metrics for each language
